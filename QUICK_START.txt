🔧 TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════════

"Connection refused"
  → Is backend running? Check: curl http://localhost:8000/health

"Cannot decode response"
  → Check model matches API response
  → Use JSONDecoder() with proper CodingKeys

"Haptics console error"
  → Don't use CHHapticPattern
  → Use HapticsService.shared.impact() instead

"Compilation errors"
  → Clean: Cmd+Shift+K
  → Delete derived data: rm -rf ~/Library/Developer/Xcode/DerivedData/*
  → Rebuild: Cmd+B
